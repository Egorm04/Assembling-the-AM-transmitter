# Assembling-the-AM-transmitter

## О проекте

Проект посвящён решению проблемы утилизации старой радиоаппаратуры (радиоприёмники, магнитолы, радиолы), которая потеряла актуальность из-за отказа от АМ-вещания в России с 2014 года. Большинство таких устройств не принимают УКВ (87.5–108 МГц) и не имеют линейного входа, поэтому отправляются на свалки, загрязняя окружающую среду пластиком и токсичными электролитами.

**Цель** — дать «вторую жизнь» старой радиоаппаратуре простыми и дешёвыми методами, сократив количество выброшенной техники.

## Решение

Основной способ — собрать **маломощный средневолновой радиопередатчик** (диапазон 526.5–1606.5 кГц) с радиусом действия не более 3 метров. Он позволяет передавать любой линейный сигнал (с телефона, компьютера, плеера) на старый АМ-приёмник без его переделки.

Дополнительно предлагается использование готовых модулей (УКВ-тюнеры, Bluetooth, MP3-плееры) для расширения функционала аппаратуры.

## Комплектующие

| Деталь | Обозначение | Примечание |
|--------|-------------|-------------|
| Транзистор | П416Б (или любой ВЧ кремниевый/германиевый) | n-p-n, маломощный |
| Конденсатор переменной ёмкости (КПЕ) | С2 | 10–200 пФ |
| Постоянный конденсатор | С1 | 0.1 мкФ (керамический) |
| Резистор | R1 | 10 кОм |
| Катушка связи | L1 | 10 витков провода ПЭВ-0.3 на ферритовом стержне |
| Антенная катушка | L2 | 40 витков поверх L1 |
| Проволочная антенна | – | отрезок провода ~1 м |
| Батарея | 9 В («Крона») | либо 5–12 В |

**Общая стоимость** — около 100 рублей (против 800–3000 руб. за готовый FM-трансмиттер).

## Сборка

Для удешевления монтаж можно выполнить на куске фанеры или картона, используя набитые медные штырьки («лужёные гвозди»). Все соединения паяные.

**Важно:** при намотке катушек строго соблюдать количество витков. Катушка связи (L1) наматывается первой (10 витков), поверх неё — антенная катушка L2 (40 витков).

## Настройка

1. Подайте питание (6 В).
2. Включите любой АМ-радиоприёмник рядом (в диапазоне средних волн).
3. Вращайте ротор КПЕ (С2) до появления «молчания» или шума — это признак генерации.
4. Перемещайте катушку L1 по ферритовому стержню для получения максимально чистого сигнала без искажений.
5. Подключите источник звука (телефон, плеер) к входу (через конденсатор С1). Радиоприёмник должен начать воспроизводить звук.

> **Если генерации нет:** переверните катушку связи (L1) на стержне — фаза обратной связи может быть неверной.

**Результаты измерений:**
- Диапазон воспроизводимых частот: **80 – 4000 Гц** (достаточно для речи и лёгкой музыки).
- Дальность: до 3 метров.

## Расширение возможностей

Помимо АМ-передатчика, можно модернизировать аппаратуру с помощью готовых модулей:

| Модуль | Функции | Пример |
|--------|---------|--------|
| УКВ-планка (аналоговая/цифровая) | Приём FM 87.5–108 МГц | Разработки радиолюбителя «Жора Минский» |
| Bluetooth-модуль | Беспроводной приём звука | HC-05, HW-199 |
| MP3-панель | USB/SD/BT/AUX/FM-тюнер | Модуль с дисплеем и пультом |

## Заключение

Предложенные методы позволяют:
- сохранить работоспособность и функциональность старой аудиоаппаратуры;
- снизить количество электронного мусора;
- избежать загрязнения почвы пластиком и электролитами.

**Преимущества АМ-передатчика:** не требует изменений в приёмнике.  
**Преимущество готовых модулей:** добавляют современные интерфейсы (BT, USB, FM).
